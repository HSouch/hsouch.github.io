GalaRP

GalaRP (Gala + Ram Pressure) is an altered version of Adrian Price-Whelan's Gala code to handle non-conservative forces, particularly those in a ram pressure event. This code is ideally suited for determining overall initial conditions for hydrodynamic wind-tunnel simulations, but can do a good job at showing the overall gravitational effects (assuming a Gunn and Gott model for acceleration).

I am currently in the process of porting GalaRP over to Rust, which I feel will allow for faster runtime calculations.

GalPRIME

I am the lead developer for GalPRIME: a pipeline that fulfills two overall functions:

  • Refines the elliptical isophote analysis code in Photutils to extract 1D surface brightness profiles of galaxies.
  • Contains a testing suite to quantify the quality of extracted profiles by adding simulated galaxies to real images and testing deviations.

This pipeline is designed to extract profiles for large populations of galaxies, and check the reliability of the results depending on structural parameters. It is designed with parallel computing in mind, and handles masking and background subtraction.

I am happy to discuss additions and alterations to the code, and would be open to collaborations on refinements and improvements. (Code Paper Submitted to ApJ)

CUBESPA

CubeSPA (Cube SPectral Analysis) is a fully-featured suite to analyze, explore, and process datacubes, with a focus on cubes from radio astronomy. It is designed to simplify the process of analyzing and plotting information on datacubes. Almost all of the plots in my upcoming papers on ALMA-JELLY-related galaxies are generated using CubeSPA. I am open to suggestions and collaborations on this code, and would be happy to hear any features that you would like to see added. The best way to do this through pull requests or issues on the linked Github page.
