Code

Almost all of my code is available for free on my Github page, but I explain and advertise some of my more helpful work here. The overwhelming majority of my work is Python related, making extensive use of Astropy, Numpy, and Photutils.

GalPRIME

I am the lead developer for a python package that fulfills two overall functions:

  • Refines the elliptical isophote analysis code in Photutils to extract 1D surface brightness profiles of galaxies.
  • Contains a testing suite to quantify the quality of extracted profiles by adding simulated galaxies to real images and testing deviations.
GalPRIME (GALaxy Profile Recovery from Images of Model Emission) is particularly designed for any researchers looking to extract profiles for large populations of galaxies, and check the reliability of the results depending on structural parameters. It is designed with parallel computing in mind, and handles masking and background subtraction.

I am happy to discuss additions and alterations to the code, and would be open to collaborations on refinements and improvements. (Code Paper in Prep)

Cutout Creator

Cutout Creator is a very simple utility to rapidly obtain cutouts of objects from a set of images, given a provided catalogue. The code also has masking and (very rudimentary) SNR cutoff functionality.

Helpful Snippets

This is a collection of convenience programs and code-snippets that I developed to make my life working with astronomical data just a little bit easier.

  • Simple Image Tester Checks if there are any corrupted FITS files in a directory by running a simple open/close check.
  • FITS Compiler Compiles a bunch of FITS files into a single large HDUList. (Beware ye with low available RAM).
  • Coverage Getter Returns a table containing the coverage information for a set of images, based on RA/DEC information.
  • Cutout Viewer Generates a visualization of a set of cutouts. Works well with the output from CutoutCreator. Similar functionality is also available in the tbridge.plotting subpackage.
  • Command Line Args Template Many times I've wanted to rapidly create a python program that can be run from the command line with optional user arguments. This is a template that allows me to get started fast.
  • TBriDGE Command Line Wrapper This is a command line wrapper to allow the user to operate TBriDGE from the command line with as few issues as possible. This allows for easy configuration to customize simulations.
